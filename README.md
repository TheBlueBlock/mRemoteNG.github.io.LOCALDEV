# [mRemoteNG.org](https://mRemoteNG.org)
mRemoteNG Website

## Requirements
- [Ruby](https://www.ruby-lang.org)
- [Bundler](https://bundler.io/)

## Building
1. Run
   ```bash
   bundle install
   bundle exec jekyll serve --incremental --config _config.yaml,_config-dev.yaml
   ```
2. Open http://localhost:4000/
